# 项目需求设计

## 项目概述
多智能体平台（ghost616），支持 LLM 模型管理、工具注册、HOOK 管理、SKILL 管理、智能体配置与执行引擎的完整系统。

## 核心需求
1. **LLM 模型管理**：支持多模型配置，统一调用接口，屏蔽不同 LLM 提供商的差异
2. **工具注册**：支持 Java/TypeScript/Python 三种类型的工具注册与管理，提供统一执行调度
3. **HOOK 管理**：基于 Java 接口的 HOOK 机制，在智能体会话生命周期各阶段自动触发
4. **SKILL 管理**：SKILL 的 CRUD 管理，支持与工具的多对多关联，供智能体引擎调用
5. **智能体配置**：可视化创建智能体，关联模型、工具、HOOK、SKILL
6. **智能体执行引擎**：对话编排、工具调用调度、上下文管理、流式推送
7. **前端交互界面**：完整的 Web 管理界面与对话交互界面

## 技术栈
- 后端：Java 17 + Spring Boot 3 + MyBatis-Plus
- 前端：React 18 + Vite + TypeScript 5
- 数据库：H2（默认）/ MySQL
- 构建：Maven（后端）+ Vite（前端）
